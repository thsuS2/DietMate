# 💰 가계부 앱 기능 기획서

## 1️⃣ 기본 기능 (필수)
> **핵심 목표:** 수입·지출 기록 및 통계

| 구분 | 기능 | 설명 |
|------|------|------|
| 예산 설정 | 월별/카테고리별 예산 | 식비, 교통비, 쇼핑 등 항목별 한도 설정 |
| 자산 관리 | 은행, 현금, 카드, 적금 등 등록 | 전체 자산 합계 및 잔액 관리 |
| 수입/지출 등록 | 단건 입력 | 날짜, 금액, 카테고리, 결제수단, 메모 |
| 반복 등록 | 반복 수입/지출 | 월세, 구독료, 급여 등 자동 생성 |
| 카테고리 관리 | 커스터마이징 가능 | 사용자가 직접 카테고리 추가/수정 가능 |
| 통계 화면 | 월별 지출 그래프 | 카테고리별/결제수단별 지출 비율 |

---

## 2️⃣ 사용자 편의 기능
> **핵심 목표:** 입력 자동화 + 직관적인 UI

| 구분 | 기능 | 설명 |
|------|------|------|
| 자동 분류 | 결제내역 OCR / 문자 인식 | 카드 문자 자동 인식 후 지출 자동 등록 |
| 태그 기능 | “데이트”, “출장” 등 | 상황별로 묶어서 통계 확인 가능 |
| 검색 및 필터 | 기간, 카테고리별 검색 | 특정 조건의 내역만 보기 |
| 달력 뷰 | 월별 달력형 기록 보기 | 특정 날짜별 소비 한눈에 확인 |
| 노트/메모 | 소비 기록에 코멘트 | 예: “친구 생일 선물” 등 |

---

## 3️⃣ 분석 및 리포트
> **핵심 목표:** 사용자의 소비 습관 인사이트 제공

| 구분 | 기능 | 설명 |
|------|------|------|
| 월간 리포트 | 자동 리포트 생성 | 이번 달 수입·지출·잔액·카테고리별 분석 |
| 예산 초과 알림 | 예산 대비 초과율 | “식비 80% 이상 사용” 알림 |
| 목표 설정 | 저축/지출 목표 | 예: “3개월 안에 100만원 저축” |
| 트렌드 분석 | 월별 변화 그래프 | 소비 추이 / 절약 효과 시각화 |
| 자산 증감 그래프 | 순자산 변화 시각화 | 월별 자산 변동 그래프 제공 |


---

## 💡 UX/UI 아이디어

- “오늘의 소비 요약” 홈 위젯  
- 카테고리별 색상 커스터마이징  
- 다크모드 / 심플 모드 / 통계 모드 전환  
- 음성 입력 (“점심 8천 원”)  
- 소비 패턴 리마인드 (예: 매주 커피 3회 이상)
