### 🔧 발견된 문제 및 수정

#### 문제 1: dailyWaterGoal undefined 처리
**위치**: `HomeScreen.js` (212, 315줄)

**문제점**:
- `settings.dailyWaterGoal`이 undefined일 때 0으로 나누기 오류 발생 가능
- 기본값이 설정되지 않은 경우 NaN 표시

**수정 내용**:
```javascript
// 수정 전
progress={todayRecord.water / settings.dailyWaterGoal}

// 수정 후
progress={settings.dailyWaterGoal > 0 ? todayRecord.water / settings.dailyWaterGoal : 0}
```

**상태**: ✅ 수정 완료

### 📝 개선 제안
1. **연속 기록 일수 계산**: 현재는 간단 버전으로 구현되어 있음. 정확한 연속 기록 계산 로직 개선 가능
2. **이번 주 통계**: `thisWeek` 계산이 단순화되어 있음. `getThisWeek()` 유틸리티 함수 사용 권장
